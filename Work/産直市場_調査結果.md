## 1. 産直市場とは

### 1.1 定義

産直市場（産地直売所）とは、生産者（農家・漁師・加工業者など）が中間流通業者を介さず、直接消費者に農産物・水産物・加工品などを販売する施設・仕組みのことである。JA（農業協同組合）が運営するファーマーズマーケットが代表的な形態であり、地産地消・生産者支援・地域活性化を目的としている。

### 1.2 代表的な事例: JAあぐりタウン げんきの郷（愛知県大府市）

| 出資者 | あいち知多農業協同組合 |
| 設立 | 平成12年3月1日 |

## 2. 産直市場の事業理念

### 2.1 アグリルネッサンス事業の柱

1. **土づくりを基本とした持続性ある有機農業の実現**
2. **生産から加工・流通・販売・消費に至る食一環システムの構築**
3. **農業を核とし、商・工・観光等とも連携した地域複合産業の形成**
4. **自然生態系（エコロジー）と人間生活（エコノミー）が調和した農業・農村文化の再生**
5. **人生100年時代をすべての人々が健やかに生きる社会（少子高齢化社会）への対応**

## 3. 産直市場の商品構成と仕入形態

### 3.1 取扱商品カテゴリ

| 商品分類 | 商品例 | 主な仕入形態 |
|----------|--------|--------------|
| 野菜・果物 | 採れたて野菜、旬の果物、山菜、野菜苗 | 委託（生産者出荷） |
| 米・穀物 | 精米、玄米、特別栽培米 | 委託 / 買取 |
| 畜産物 | 鶏卵、知多牛、名古屋コーチン、愛ポーク | 買取 / 委託消化 |
| 水産物 | 地魚（生鮮）、干物、冷凍水産加工品 | 買取（テナント形式） |
| 花き | 切花、花苗、園芸資材 | 委託 |
| 加工品 | ジャム・ジュース・漬物・豆腐・銘酒 | 委託消化 / 買取 |
| 惣菜・弁当 | でき立て惣菜、デリカ、だんご | 委託消化（自社加工） |

### 3.2 生産者情報の管理

- **全品に生産者名を表示**（トレーサビリティ確保）
- 地元生産者が毎日ていねいに出荷
- 食味重視の品種・季節限定品も取り扱い
- こだわりの「自信作」として提供

## 4. 産直市場の運営仕組み

### 4.1 基本的な業務フロー

```
【生産者】
  ↓ 登録申請・出荷者登録
【産直市場（運営側）】
  ↓ 会員登録・出荷許可
【生産者】
  ↓ 農産物の収穫・梱包・値付け・バーコード発行
  ↓ 店舗への搬入・陳列
【販売（POS・レジ）】
  ↓ 消費者による購入
  ↓ POSデータ記録（出荷者別・品目別）
【精算処理】
  ↓ 売上集計（日次・月次）
  ↓ 販売手数料控除（一般的に10〜30%）
  ↓ 生産者への売上精算（月次振込）
```

### 4.2 生産者（出荷者）の登録・管理

| 管理項目 | 内容 |
|----------|------|
| 出荷者登録 | 氏名・住所・口座情報・品目登録 |
| 出荷者番号 | 固有ID（バーコード・QRコードに紐付け） |
| 取扱品目 | 販売許可品目の登録・審査 |
| 規約遵守 | 農薬使用基準・表示ルール・品質基準の遵守 |
| 手数料率 | 品目・販売形態によって設定（委託/買取で異なる） |

### 4.3 商品管理（出荷〜販売）

| プロセス | 詳細 |
|----------|------|
| 商品登録 | 生産者が商品情報（品名・価格・数量・産地）を登録 |
| バーコード発行 | 出荷者番号・品目コード・価格を含むバーコード印刷 |
| 搬入 | 開店前に生産者が直接店舗に搬入・陳列 |
| 在庫管理 | POSシステムで入荷・販売数を管理 |
| 廃棄処理 | 閉店時に売れ残り品の回収（生産者が引き取り） |

### 4.4 精算・会計処理

| 項目 | 内容 |
|------|------|
| 精算サイクル | 月次精算が主流（週次対応のシステムも存在） |
| 精算方式 | 売上金額 - 販売手数料 = 支払額 |
| 手数料率 | 品目別・出荷者別に設定（一般的に10〜30%） |
| 精算明細 | 品目別販売数・金額・手数料の明細発行 |
| 振込処理 | 指定口座への振込 |

## 6. 産直市場の仕入形態の分類

### 6.1 委託販売（委託）

生産者が商品の所有権を保持したまま、産直市場が販売代行を行う形態。

| 項目 | 内容 |
|------|------|
| 所有権 | 生産者が保持（販売完了まで） |
| リスク | 売れ残りは生産者の損失（引き取り義務） |
| 精算タイミング | 販売後に精算 |
| 手数料 | 売上の一定割合（例：15〜25%） |
| 主な商品 | 野菜・果物・花き・加工品 |
| 特徴 | 産直市場の主流形態。生産者が価格決定権を持つ |

### 6.2 委託消化（委託消化仕入）

百貨店・大型小売店で広く用いられる形態。商品が実際に販売された時点で仕入が確定する。

| 項目 | 内容 |
|------|------|
| 所有権 | 販売されるまでメーカー・サプライヤー側に帰属 |
| リスク | 在庫リスクはサプライヤー側が負担 |
| 精算タイミング | 販売実績に基づき確定・精算 |
| 手数料 | 掛率に基づく（例：上代の60〜75%を仕入価格とする） |
| 主な商品 | アパレル・加工食品・ギフト品 |
| 特徴 | 小売店はリスクが低い。サプライヤーは在庫管理が重要 |

### 6.3 買取仕入（買取）

小売店が商品の所有権をメーカー・生産者から購入する形態。

| 項目 | 内容 |
|------|------|
| 所有権 | 仕入時に小売店へ移転 |
| リスク | 在庫リスクは小売店が負担 |
| 精算タイミング | 仕入時に確定（請求・支払） |
| 価格 | 仕入価格（卸価格）で購入 |
| 主な商品 | 精肉・水産物・米・日配品 |
| 特徴 | 小売店が価格決定権を持つ。仕入ロット・数量管理が重要 |

## 7. 産直市場の開設・運営のポイント

### 7.1 施設・設備

- 適切な売場面積（500〜2,000㎡以上推奨）
- 冷蔵・冷凍設備（生鮮品・畜産・水産向け）
- 搬入口・バックヤードの確保
- POSレジ・バーコードスキャナーの導入
- 生産者ラベル印刷機の配備

### 7.2 生産者の確保・組織化

- 地元農家・漁業者・加工業者の組織化
- 出荷基準・品質基準の策定
- 出荷者規約・販売手数料の明確化
- 定期的な出荷者説明会・研修の実施

### 7.3 品揃えの充実

- 地元産品の優先確保
- 季節商品の計画的な品目管理
- 欠品対策（補完仕入・買取との組み合わせ）
- 加工品・特産品との差別化

### 7.4 情報発信・集客

- SNS・公式サイトによる情報発信
- ポイントカード・アプリによる顧客管理
- イベント・体験活動との連携
- 地域ブランド（産地名・生産者名）の活用

## 8. 産直市場の課題と対策

| 課題 | 内容 | 対策 |
|------|------|------|
| 品揃えの波動 | 天候・季節による出荷量の変動 | 複数品目・複数生産者による補完 |
| 鮮度管理 | 生鮮品の廃棄ロス | 値下げ機能・消費期限管理の自動化 |
| 生産者管理 | 出荷者の高齢化・減少 | 新規出荷者の発掘・育成支援 |
| 精算業務 | 手作業による精算ミス | システム化による自動精算 |
| 表示コンプライアンス | 農薬・産地偽装リスク | 出荷者登録審査・トレーサビリティ管理 |

## 10. まとめ

産直市場の作り方のポイントは以下の通りである。

1. **地域農業・生産者との強固なネットワーク構築**
2. **委託・委託消化・買取の適切な組み合わせによる商品構成**
3. **POSシステムと出荷者管理・精算システムの一体化**
4. **トレーサビリティ（生産者情報の可視化）の確保**
5. **食・農・観光・文化の複合施設としての付加価値創出**
6. **SNS・デジタルツールを活用した継続的な情報発信と集客**
7. **地域ブランド・地産地消の取り組みによる差別化**

産直市場のシステム化においては、委託・委託消化・買取それぞれの仕入形態に対応した精算ロジックを構築し、生産者（出荷者）への透明性の高い精算業務を実現することが最重要課題となる。
